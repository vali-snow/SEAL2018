User guide

Introduction

This is a technical document intended to give an overview on how our application is to be used, and the features it has.

Notification WorkFlow

Once we accessed the app, a notification alert will appear on the top of the screen where we are asked if we allowed to receive notifications directly in the browser.

If we choose to allow notifications, a pop-up will show up where we check which types of notifications we want to subscribe then save the options by clicking the save button.

After saving the settings, we will return to the main page where the last 6 allergies will appear.

If there is a new post about the type of allery se subscribe on, we will receive a notification in the browser.

Navigating Allergies WorkFlow

If we want to find quickly a specific allergy, we can use the search field by typeing keywords.

Once we accessed an allergy, we will be redirect to the allergy page wher relative informations will show up like: short description, symptoms, prevention methods, treatment and medication. Also a graphical representation of the number of people affected over the ourse of 3 years.

We can share this information with our group of friends by pressing the share on facebook button.

We are able to select if we want to share this allergy infos only with a friend or a certain group or with all of our friends on social network.
