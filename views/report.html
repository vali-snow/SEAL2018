General Design Concepts

The page design can have a significant impact on the success of our communication, on it's usability and persuasiveness.

That is why we will try to focus on reader-centered pages, facilitating the understanding of information & the ability to find it with ease.

User Interaction

While using our application, the user will have the following features available:

  • Accept & Receive Notifications
  • Search for allergies
  • See details & stats for a specific allergy

Source Code

The tool we chose to use for code hosting is Github .

We have created a public repository at: SEAL2018

And we will be using the suggested workflow for our features:

  1. Creating a new branch for the feature.
  2. Making as many commits as necessary.
  3. Merging the new branch into the main one.

Team Coordination

We have decided to use Trello for our Project Tracking Software.

We have created a different board for each stage of the project.

Our workflow aim is to be agile (with a twist), thus our boards we have the following layout:

  • Guidelines
  • Backlog
  • To Do
  • Doing
  • Done

Finally, as a communicating hub for our team, we chose Slack .

We improved it by integrating Trello & Github into it.
